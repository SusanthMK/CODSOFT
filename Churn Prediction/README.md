# 📊 Customer Churn Prediction

## 📌 Overview
This project predicts whether a customer is likely to leave (churn) using machine learning techniques. It helps businesses take proactive actions to retain customers.


## 🔧 Techniques Used
- Data preprocessing & encoding
- Handling class imbalance
- XGBoost Classification
- Threshold tuning



## 📊 Results
- Accuracy: ~85–90%
- High ROC-AUC score
- Improved churn detection using optimized threshold



## 📈 Output
- Confusion Matrix
- ROC Curve
- Feature Importance Visualization



## 🚀 How to Run
1. Open `Churn_Prediction.ipynb`
2. Run all cells
3. Analyze predictions and visual outputs

# 💳 Fraud Detection

## 📌 Overview
This project aims to detect fraudulent financial transactions using machine learning. It focuses on handling **imbalanced datasets** and building a robust classification model.


## 🔧 Techniques Used
- Data preprocessing & cleaning
- Handling class imbalance
- XGBoost (Gradient Boosting)


## 📊 Results
- High ROC-AUC score
- Strong fraud detection performance
- Reliable classification of rare fraud cases


## 📈 Output
- Confusion Matrix
- Precision-Recall Curve
- Feature Importance Graph



## 🚀 How to Run
1. Open `fraud_model.ipynb`
2. Run all cells
3. View evaluation metrics and visual outputs

# 📩 Spam SMS Detection

## 📌 Overview
This project focuses on detecting spam messages using Natural Language Processing (NLP) techniques. The model classifies SMS messages as either **Spam** or **Not Spam**.

---

## 🔧 Techniques Used
- Text preprocessing (lowercasing, removing punctuation & numbers)
- TF-IDF Vectorization
- Machine Learning Models:
  - Naive Bayes
  - Logistic Regression
  - Support Vector Machine (SVM)

---

## 📊 Results
- High accuracy classification
- Best model selected after comparison
- Real-time prediction capability

---

## 📈 Output
- Confusion Matrix
- Sample Predictions

---

## 🚀 How to Run
1. Open `spam_model.ipynb`
2. Run all cells
3. Test custom messages using prediction function
